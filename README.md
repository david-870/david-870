# Hi, I'm David 👋

I'm a self-taught developer based in **Ibadan, Nigeria**, on a mission to turn curiosity into code — and code into products people actually use.

I've been building for almost three years now, and in that time I've gone from writing my first HTML tag to shipping real websites for real clients. I don't just code — I build systems.

---

## 🚀 What I'm Working On

- 🧊 **[Adodo Frozen Food](https://github.com/david-870/adodo-frozen-food)** — Full-stack storefront for a frozen food shop in Ilorin. Product catalog, admin panel, Node.js API, WhatsApp checkout, and a support chatbot. *Live client project.*
- 🛍️ **BaloTech Gadget Hub** — A full e-commerce storefront for a physical gadget store in Nigeria. Multi-page product catalog, cart system, admin stock management, and WhatsApp checkout. Built with pure HTML, CSS and JavaScript.
- ☀️ **Solar Client Website** — Currently building a professional website for a solar energy business client. A real-world project geared towards bringing solar energy to everyone.
- 💪 **Mastering JavaScript & React** — Going deep on JS fundamentals before fully diving into React.

## 📚 Learning Journey

I started from scratch — no CS degree, no bootcamp, just consistency and curiosity. Over the past year I've worked through:

- Frontend Mentor challenges to sharpen my HTML and CSS
- Building real projects for real people
- Writing JavaScript that actually does things — carts, DOM manipulation, APIs, and admin panels
- Backend basics with **Node.js & Express** — auth, JSON storage, deployment
- Currently levelling up with React

I believe the fastest way to learn is to build things that matter, so every project I take on solves a real problem for a real person.
